平台公约
以下条例适用于本平台所有用户
  1. 严禁发表反党反政府的言论，或作出诋毁党和国家的行为；
  2. 严禁传播低俗、涉黄、裸露、涉政、歧视、性暗示等内容；
  3. 严禁传播毒品，展示枪支管制刀具，公然在镜头前抽烟喝 酒、赌博、自残等暴力血腥内容；
  4. 严禁进行各类广告宣传，恶意发布广告，宣传传销机构、 哄骗用户加入传销组织，或本人组织传销等；
  5. 严禁以文字、图片、视频等任何形式宣传违规内容；
  6. 平台已接入鉴黄系统和后台人工审核，对于违规的内容进行实施删除拦截；
  7. 对以上行为实施从严标准，所有用户，如果言论和行为有违反上述内容的，将会视情节严重予以24小时、72小时、30天或永久封号处理；
  8. 受理所有问题时，平台绝不以任何理由向用户收费，如遇收费均属欺诈行为，欢迎大家监督举报。
